Обзор программы Type it Easy

Программа позволяет легко и просто вводить с клавиатуры любые символы. Выберите один из готовых наборов или настройте каждую клавишу по своему вкусу.

Чтобы напечатать нужный символ, прижмите правый Alt (AltGr), а затем нажмите соответствующую клавишу. Например, одновременное нажатие AltGr и клавиши «C» напечатает символ ©, а нажатие AltGr + «E» вставит символ евро: €.

При печати текста на русском языке, очень неудобно вводить многие символы, например доллар $ — приходитя постоянно переключать раскладку. Используя Type it Easy, просто нажмите AltGr + «4» и проблема с долларами решена.

Нажатие AltGr одновременно с шифтом напечатает альтернативный символ, указанный в верхнем правом углу клавиши. Например, AltGr + Shift + «P» даст символ «параграф»: §.
